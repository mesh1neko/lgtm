Keywords that act as a +1, one keyword per line. Examples: :+1:, LGTM, :thumbsup:
Keywords are case-insensitive.
TIP: Emoji like :+1: or :thumbsup: can be used as keywords.


Keywords that act as a -1, one keyword per line. Examples: :-1:, REJECTED, :thumbsdown:
Keywords are case-insensitive.
TIP: Emoji like :-1:, :poop: or :scream_cat: can be used as keywords.



Disable merge button if there are not enough positive reviews


The minimum amount of positive reviews needed to merge
For example if set to 2, a pull request with two +1s and one -1 (ie. +1 overall) will have a disabled merge button


Disable merge button if pull request is marked as work-in-progress.
Enabling this option will also grey out the pull request in the pull request list.


Keywords in pull request title to signify WIP. One per line.
Keywords are case-insensitive



Add a badge with the vote count to the favicon on the pull request details page.
If you have another extension that changes the favicon (eg. CI status color), you can disable this feature.
